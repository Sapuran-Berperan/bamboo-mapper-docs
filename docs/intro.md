# Pengenalan

**Bamboo Mapper** adalah aplikasi Android untuk memetakan lokasi bambu di Kecamatan Sapuran, Kabupaten Wonosobo, Jawa Tengah. Aplikasi ini dikembangkan sebagai bagian dari program KKN-PPM UGM Periode IV 2025.

## Tentang Aplikasi

Bamboo Mapper memungkinkan pengguna untuk:

- Menandai lokasi rumpun bambu pada peta interaktif
- Mencatat informasi detail seperti jenis bambu, jumlah, dan pemilik lahan
- Menghasilkan QR code untuk setiap titik lokasi
- Mengakses data bambu melalui pemindaian QR code

## Arsitektur Sistem

Sistem Bamboo Mapper terdiri dari dua komponen utama:

1. **Aplikasi Mobile (Android)** - Dibangun menggunakan Flutter untuk antarmuka pengguna dan pemetaan
2. **Backend API** - Dibangun menggunakan Go dengan framework Chi untuk mengelola data dan autentikasi
